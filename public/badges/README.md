- `oracle-genai.png` - Oracle Cloud Infrastructure Generative AI badge
- `oracle-k8s.png` - OCI Kubernetes Engine Specialist badge
- `cpp.png` - C++ Programming badge (HPE Data Science Institute)
- `excel-ds.png` - Excel for Applied Data Science badge (HPE Data Science Institute)
- `excel-powerbi.png` - Excel & Power BI badge (HPE Data Science Institute)

The images will be automatically loaded and displayed in the Badges section of the portfolio.
